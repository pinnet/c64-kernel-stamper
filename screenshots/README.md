# App Screenshots

This directory contains screenshots for the C64 Kernel Stamper Progressive Web App (PWA) listing.

## Required Screenshots

1. **desktop.png** (1280x720)
   - Wide format for desktop/tablet view
   - Show the full interface with a ROM loaded
   - Include the file browser, preview, and configuration panel

2. **mobile.png** (540x720)
   - Narrow format for mobile view
   - Show the responsive mobile layout
   - Focus on the preview and key controls

## How to Capture Screenshots

1. Open the app in a browser
2. Load a sample ROM file
3. Make some edits to show functionality
4. Use browser DevTools to set exact viewport dimensions
5. Capture screenshots using:
   - Browser's built-in screenshot tool (Firefox DevTools)
   - OS screenshot utility
   - Browser extension like "Awesome Screenshot"

## Tips

- Use actual content (loaded ROM with edits)
- Show the C64 preview with custom text/colors
- Ensure good contrast and readability
- Avoid personal information in screenshots
