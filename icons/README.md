# App Icons

This directory contains icon files for the C64 Kernel Stamper Progressive Web App (PWA).

## Required Icon Sizes

Create PNG icons in the following sizes:
- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Icon Design Guidelines

- Use the C64 theme with the monitor/computer symbol from the header
- Background: Dark blue gradient (#0a0e1a to #141b2e)
- Primary color: #0088FF (Commodore blue)
- Include rounded corners for better appearance on different platforms
- Ensure the icon is recognizable at small sizes

## How to Generate Icons

You can use tools like:
- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Adobe Photoshop/Illustrator
- Figma with export settings

Name the files as `icon-{size}.png` (e.g., `icon-192x192.png`)
